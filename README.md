My solutions for the Advent of Code 2017

**Table of Contents**

## Prerequisites

- Python 3 (https://www.python.org/)

## Installation

```bash
git clone https://github.com/erictleung/advent2017.git
cd advent2017
python setup.py install
```

## Testing

```bash
git clone https://github.com/erictleung/advent2017.git
cd advent2017
python -m unittest
python -m unittest -v # See tests being run
```
